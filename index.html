Bem-vindo(a) ao Linux From Scratch!

O Linux From Scratch (LFS) é um projeto que fornece a você instruções passo-a-passo para construir o teu próprio sistema personalizado GNU/Linux, inteiramente a partir do código fonte.

Atualmente, a organização Linux From Scratch consiste dos seguintes subprojetos:

  • LFS :: Linux From Scratch é o livro principal, a base a partir da qual todos os outros projetos são derivados.
  • BLFS :: Beyond Linux From Scratch ajuda você a estender a tua instalação finalizada do LFS em um sistema mais personalizado e usável.
  • ALFS :: Automated Linux From Scratch fornece ferramentas para automatizar e gerenciar construções do LFS e do BLFS.
  • Dicas :: O projeto Dicas é uma coleção de documentos que explicam como melhorar o teu sistema LFS de maneiras que não estão inclusas nos livros LFS ou BLFS.
  • Remendos :: O projeto Remendos serve como um repositório central para todos os remendos úteis para um(a) usuário(a) do LFS.
  • Guia do(a) Editor(a) do LFS :: Um documento que descreve o processo de desenvolvimento do LFS.
  • Museu :: Cópias das versões antigas do LFS e do BLFS.
