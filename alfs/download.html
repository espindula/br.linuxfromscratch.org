Baixar jhalfs

Obter jhalfs

jhalfs é mantido como um lançamento contínuo. Ele pode ser obtido a partir do repositório git:

git clone https://git.linuxfromscratch.org/jhalfs.git

Pacotes exigidos para jhalfs

Antes que consiga analisar o livro LFS em busca de comandos, o jhalfs precisa ter uma cópia local do fonte XML do livro. Atualmente, jhalfs depende do git para garantir que tenha a versão mais atualizada do livro LFS. Existem instruções de construção no BLFS.

Para extrair os comandos a partir do livro LFS, o jhalfs depende do xsltproc oriundo do pacote libxslt e das dependências dele. Existem instruções de construção no BLFS.

O DTD do XML do DocBook e folhas de estilo XSL do DocBook agora são necessárias para perfilar o livro. Existem instruções de construção para DTD do XML do DocBook e para folhas de estilo XSL do DocBook no BLFS.

Se você quiser que o jhalfs baixe automaticamente os pacotes fonte que o LFS usa, você precisará ter o wget instalado. Existem instruções de construção no BLFS.

Por último, porém não menos importante, jhalfs depende do sudo para executar instruções privilegiadas. Novamente, instruções de construção podem ser encontradas no BLFS. O arquivo README tem alguns conselhos acerca de como configurar /etc/sudoers.
