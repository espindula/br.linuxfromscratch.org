Como Contribuir

Torne-se um espelho

Você pode ajudar o projeto LFS fornecendo um espelho HTTP ou FTP. Os espelhos HTTP fornecem o sítio do LFS, livros, remendos e dicas. Os espelhos FTP podem fornecer vários serviços, incluindo pacotes LFS e pacotes BLFS. Os requisitos para os vários espelhos estão listados abaixo. Se você gostaria de se tornar um espelho, por favor, configure-o e envie uma mensagem eletrônica com os dados solicitados abaixo para o mantenedor do sítio, webmaster@linuxfromscratch.org. Obrigado(a).

Requisitos

Aqui está uma lista de requisitos para os vários espelhos. Todos os espelhos exigem que você use rsync para sincronizar teus espelhos com nossos servidores principais e um aplicativo cron para agendá-la para você.

Você pode encontrar hospedagem web do Linux barata ou um servidor dedicado inteiro para teu espelho na pesquisa de hospedagem na web.

Espelho HTTP

  • Pelo menos 5 GB de espaço em disco.
  • Servidor web Apache ou equivalente com as seguintes opções habilitadas:
    • Indexes
    • FollowSymlinks
    • IncludesNOEXEC (requires mod_include.so)
  • Se você quiser usar o Nginx em vez do Apache, as seguintes opções deveriam ser usadas:
    • ssi on; (requires ngx_http_ssi_module)
    • disable_symlinks off;
    • autoindex on;

Para configurar o espelho, use a seguinte instrução rsync em um conjunto de comandos sequenciais cron:

/usr/bin/rsync -lprt --delete www.linuxfromscratch.org::lfs-website /srv/www/lfs

Espelhos FTP do LFS e BLFS

  • Pelo menos 300 GB de espaço em disco.
  • O caminho para o espelho sendo "/pub/lfs/" e "/pub/blfs/" se possível.

Para configurar os espelhos FTP, use as instruções rsync em um conjunto de comandos sequenciais cron:

/usr/bin/rsync -lprt --delete rsync.osuosl.org::lfs /srv/ftp/LFS
/usr/bin/rsync -lprt --delete rsync2.osuosl.org::blfs /srv/ftp/BLFS

Ainda interessado(a)?

Se você ainda estiver interessado(a) em se tornar um espelho, por favor, envie as informações a seguir por mensagem eletrônica para a lista apropriada ou para o webmaster.

  • Local (Cidade e País)
  • Nome do dispositivo e Endereço IP
  • Largura de banda
  • Nome de contato e Endereço válido de correio eletrônico
