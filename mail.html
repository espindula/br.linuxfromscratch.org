Listas de Discussão do LFS

Preâmbulo

Antes que você poste nas listas de discussão, por favor, verifique se tua resposta já foi respondida nas Perguntas Frequentes, na página de pesquisa ou por outros métodos na página de suporte. Se você não entender uma resposta nas Perguntas Frequentes ou gostaria de mais informações, envie uma mensagem eletrônica para o(a) mantenedor(a) das Perguntas Frequentes.

Se você tiver terminado com o livro LFS, use o blfs-support. Por favor, leia as regras da netiqueta antes de postar!

Você precisa se inscrever em uma lista antes de postar.

Etiqueta importante da lista de discussão

  • Não poste no topo. As postagens de resposta deveriam estar ao final da mensagem ou intercaladas na mensagem, se existirem vários pontos.
                 R. Porque atrapalha a ordem na qual as pessoas normalmente leem o texto.
             P. Por que postar no topo é uma coisa tão ruim?
             R. Postagem no topo.
             P. Qual é a coisa mais chata na mensagem eletrônica?
  • Ao responder, corte o texto não relevante, incluindo os rodapés da lista de discussão, se o teu cliente de mensagens eletrônicas não fizer isso automaticamente.
  • Poste usando texto simples, não HTML.

Descrição das Listas de Discussão

lfs-announce
A lista lfs-announce é uma lista moderada e de baixo tráfego para aqueles(as) que querem notificação de novos lançamentos estáveis sem o maior tráfego da lfs-dev.
lfs-book
A lista lfs-book é usada para coordenar a manutenção do livro LFS. O tráfego nela é principalmente mensagens de commit do Trac e Git. É importante que todas as discussões do desenvolvimento de interesse para os(as) usuários(as) do Livro ocorram na lfs-dev, não aqui. Essa é uma lista moderada.
lfs-dev
A lista lfs-dev é somente para discussão do desenvolvimento do livro LFS.
lfs-support
A lista lfs-support é o lugar para se obter ajuda seguindo as instruções no lançamento estável do livro LFS em si. Se você tiver terminado com o livro LFS, por favor, use a blfs-support.
lfs-admin
A lista lfs-admin é uma lista moderada usada para coordenar dicas, remendos e o sítio. O tráfego nela é principalmente mensagens de commit do Trac e Git.
blfs-book
A lista blfs-book é usada para coordenar a manutenção do livro BLFS. O tráfego nela é principalmente mensagens de commit do Trac e Git. É importante que todas as discussões do desenvolvimento de interesse para os(as) usuários(as) do Livro ocorram na blfs-dev, não aqui. Essa é uma lista moderada.
blfs-dev
A lista blfs-dev é somente para discussão do desenvolvimento do livro BLFS.
blfs-support
A lista blfs-support é para o livro BLFS e além. Ela inclui solicitações de ajuda seguindo o Livro BLFS, solicitações de ajuda com pacotes que (ainda) não estão no Livro LFS ou no Livro BLFS, e outras solicitações e discussões concernentes a software que pode ser instalado em um sistema LFS. Essa lista é de tráfego relativamente alto.
alfs-discuss
A lista alfs-discuss é para discussão do desenvolvimento do ALFS.

Arquivamentos

Cada lista de discussão é arquivada e pode ser navegada online. Os arquivamentos estão sempre atualizados. Os links para os arquivamentos estão disponíveis por intermédio da página Arquivamento de cada lista.

Os arquivamentos estão divididos por mês. Se você não quiser apenas navegar, mas estiver procurando informações específicas, será melhor pesquisar nos arquivamentos usando o mecanismo de pesquisa ou a função de pesquisa de arquivamentos de lista.

Inscrever-se

A maneira mais fácil de se inscrever é por meio da página "https://lists.linuxfromscratch.org/sympa/info/<lista>". Basta selecionar "Inscrever-se"

O software da lista de discussão permite que você se logue, mas só é exigido para personalizar as opções da lista ou para cancelar a inscrição. Ao logar-se pela primeira vez, selecione “Primeiro login?”. Você receberá uma mensagem com um link onde poderá configurar tua senha. Observe que isso só precisa ser feito uma vez e a tua senha é válida para todas as listas de discussão do LFS.

Cancelar inscrição

Para cancelar a inscrição em uma lista, acesse a página de informações da lista e selecione "Cancelar inscrição".

Filtros

Cada mensagem eletrônica enviada pelo software da lista de discussão em linuxfromscratch.org terá o nome da lista entre colchetes no início da linha Assunto:. Portanto, esse cabeçalho para uma mensagem eletrônica para lfs-dev será parecido com [lfs-dev] Algum título e uma mensagem eletrônica para blfs-support será parecida com [blfs-support] Outro título.

Nós sugerimos que você use o cabeçalho Assunto: para filtrar mensagens do LFS recebidas em tuas pastas de mensagens.

Resumos

Todas as listas tem o modo resumo disponível, que pode ser configurado depois que um(a) usuário(a) tenha se inscrito em uma lista. Estar no modo resumo fará com que você pare de receber mensagens individuais à medida que elas são postadas na lista e, em vez disso, receba uma mensagem eletrônica periódica contendo todas as mensagens postadas na lista desde o período anterior.

Essa definição pode ser configurada por meio do link "Opções de Assinante" em cada lista.

Modo férias

Se você vai se ausentar por um tempo ou deseja parar de receber mensagens das listas, mas não quer cancelar a inscrição, você pode mudar para o modo férias. Isso tem o mesmo efeito que o cancelamento da assinatura, mas sem a necessidade de passar pelo processo de cancelamento e, posteriormente, pelo processo de assinatura novamente.

Essa definição pode ser configurada por meio de um formulário no link "Opções de Assinante" em cada lista.
