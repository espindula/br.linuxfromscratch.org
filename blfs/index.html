O que é Beyond Linux From Scratch?

Beyond Linux From Scratch (BLFS) é um projeto que continua onde o livro LFS termina. Ele auxilia os(as) usuários(as) no desenvolvimento dos sistemas deles(as) de acordo com as necessidades deles(as), fornecendo uma ampla gama de instruções para instalação e configuração de vários pacotes no topo de um sistema LFS básico.

Por que eu precisaria de um sistema BLFS?

Se você está se perguntando por que precisaria de um sistema BLFS ou não sabe o que é LFS, então você não quer estar aqui ainda — você deveria ir até a página inicial do projeto LFS onde tudo será explicado.

O que eu consigo fazer com meu sistema BLFS?

Quase tudo! Um sistema LFS está preparado para se tornar um sistema que atenda a qualquer necessidade que você tenha. BLFS é o livro que leva você ao teu próprio caminho personalizado. Você poderia construir uma estação de trabalho de escritório, um desktop multimídia, um roteador, um servidor ou todos os itens acima! E a melhor parte é que você só instala o que precisa.
