Notícias do LFS

Lançamento LFS-12.2
Jamenson Espindula - 2024/09/01

A comunidade brasileira do Linux From Scratch anuncia o lançamento do LFS Versão 12.2.

As principais mudanças incluem atualizações do conjunto de ferramentas para binutils-2.43.1, glibc-2.40 e gcc-14.2.0. No total, 45 pacotes foram atualizados desde o lançamento mais recente e extensas atualizações no texto foram feitas ao longo do livro para melhorar a legibilidade. O núcleo Linux foi atualizado para a versão 6.10.5.

No geral, aconteceram 146 submissões no LFS desde a versão estável anterior do livro.

Você consegue ler o livro online, ou baixar para ler localmente.

Você consegue ler a versão systemd do livro online em LFS-systemd, ou baixar-systemd para ler localmente.

Por favor, direcione quaisquer comentários acerca desse lançamento para a equipe de desenvolvimento do LFS em lfs-dev@lists.linuxfromscratch.org (somente em inglês). Por favor, observe que o registro na lista de discussão (inglesa) lfs-dev é exigido para evitar lixo eletrônico.

Lançamento LFS-12.0
Jamenson Espindula - 2023/09/06

A comunidade brasileira do Linux From Scratch anuncia o lançamento do LFS Versão 12.0.

As principais mudanças incluem atualizações do conjunto de ferramentas para binutils-2.41, gcc-13.2.0 e glibc-2.38. No total, 38 pacotes foram atualizados desde o último lançamento. Extensas atualizações para o texto ao longo do livro para melhorar a legibilidade. O núcleo Linux foi atualizado para a versão 6.4.12.

outras mudanças significativas incluem:

  • Adição de libxcrypt como um pacote separado. Anteriormente, esse pacote estava incluído no glibc.
  • Para a versão sysV do livro, o udev agora é extraído diretamente a partir do tarball do systemd. Essa mudança também exige construir os módulos Python jinja e markupsafe.
  • O novo pacote pkgconf agora é usado em vez de pkg-config porque os pacotes posteriores não mais são mantidos.
  • O novo módulo Python flit-core foi adicionado como uma nova dependência exigida para o módulo wheel.

No geral, ocorreram 234 commits no LFS desde a versão estável anterior do livro.

Você consegue ler o livro online, ou baixar para ler localmente.

Você consegue ler a versão systemd do livro online em LFS-systemd, ou baixar-systemd para ler localmente.

Por favor, direcione quaisquer comentários acerca deste lançamento para a equipe de desenvolvimento do LFS em lfs-dev@lists.linuxfromscratch.org (somente em inglês). Por favor, observe que o registro na lista de discussão (inglesa) lfs-dev é exigido para evitar lixo eletrônico.
