Lançamento do LFS-12.0
Bruce Dubbs - 2023/09/01

A comunidade Linux From Scratch anuncia o lançamento da versão 12.0 do LFS.

As principais mudanças incluem atualizações do conjunto de ferramentas para binutils-2.41, gcc-13.2.0 e glibc-2.38. No total, 38 pacotes foram atualizados desde o último lançamento. Extensas atualizações no texto ao longo do livro para melhorar a legibilidade. O núcleo Linux foi atualizado para a versão 6.4.12.

outras mudanças significativas incluem:

  • Adição de libxcrypt como um pacote separado. Anteriormente, esse pacote era incluído com a glibc.
  • Para a versão sysV do livro, o udev agora é extraído diretamente a partir do tarball do systemd. Essa mudança também exige construir os módulos Python jinja e markupsafe.
  • O novo pacote pkgconf agora é usado em vez de pkg-config porque os pacotes posteriores não mais serão mantidos.
  • O novo módulo Python flit-core foi adicionado como uma nova dependência exigida para o módulo wheel.

No geral, aconteceram 234 commits no LFS desde a versão estável anterior do livro.

Você consegue ler o livro (escrito em português) online ou baixar para ler localmente.

Você consegue ler a versão systemd do livro (escrito em português) online em LFS-systemd ou baixar-systemd para ler localmente.

Lançamento do LFS-11.0
Bruce Dubbs - 2021/09/01

A comunidade Linux From Scratch anuncia o lançamento da versão 11.0 do LFS. A razão para o incremento do número do lançamento principal é que essa versão não mais usa o sistema de "usuário(a) dividido(a)". Ou seja, como a maioria das distribuições atuais, /bin é um link simbólico para /usr/bin. Da mesma forma, /lib e /sbin são ambos links simbólicos para as contrapartes /usr deles.

Mudanças importantes adicionais incluem atualizações do conjunto de ferramentas para gcc-11.2.0, glibc-2.34 e binutils-2.37. O núcleo Linux também foi atualizado para a versão 5.13.12.

No total, 40 pacotes foram atualizados desde o último lançamento. Mudanças no texto também foram feitas ao longo do livro.

Você consegue ler o livro (escrito em inglês) online ou baixar para ler localmente.

Em coordenação com esse lançamento, uma nova versão do LFS usando o pacote systemd também está sendo lançada. Esse pacote implementa o estilo systemd mais recente de inicialização e controle do sistema e é consistente com o LFS na maioria dos pacotes.

Você consegue ler a versão systemd do livro (escrito em inglês) online em LFS-systemd ou baixar-systemd para ler localmente.

Por favor, direcione quaisquer comentários acerca desse lançamento para a equipe de desenvolvimento do LFS em lfs-dev@lists.linuxfromscratch.org (somente em inglês). Por favor, observe que o registro na lista de discussão (inglesa) lfs-dev é exigido para evitar lixo eletrônico.

Bruce Dubbs - 2021/08/25

A comunidade Linux From Scratch anuncia o lançamento da versão 11.0-rc3 do LFS. Ela é uma versão preliminar do LFS-11.0.

Essa versão do livro Linux From Scratch foi lançada devido a uma vulnerabilidade crítica de segurança no openssl. Essa vulnerabilidade foi corrigida no LFS 11.0-rc3 atualizando-se para a versão mais recente: openssl-1.1.1l (versão 1.1.1 L minúsculo). O uso do LFS 11.0-rc1 ou LFS 11.0-rc2 é considerado arriscado e inseguro.

Em adição ao acima, nós aproveitamos a oportunidade para atualizar vários outros pacotes dos lançamentos recentes de -rc para as versões estáveis mais recentes: e2fsprogs, meson, o núcleo Linux, util-linux e libcap.

Essa versão do livro Linux From Scratch mudou de uma configuração /usr dividida para uma /usr mesclada. Isso coloca o LFS em uma configuração semelhante à das distribuições mais recentes.

As principais mudanças incluem atualizações do conjunto de ferramentas para binutils-2.37 e glibc-2.34. No total, 38 pacotes foram atualizados desde o último lançamento. Mudanças no texto também foram feitas ao longo do livro. O núcleo Linux também foi atualizado para a versão 5.13.12.

Nós incentivamos todos(as) os(as) usuários(as) a lerem essa versão do livro e testar as instruções para que possamos tornar o lançamento final o melhor possível.
